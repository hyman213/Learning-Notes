# Maven

--------------

## 安装

* 下载

  > http://maven.apache.org/download.cgi

* 解压

## 常用配置

* 配置国内镜像

  * 更换conf/setting.xml为对应配置

* 更改本地仓库位置

  ```xml
  <localRepository>D:\java\rep</localRepository>
  ```

## 使用

* 构建maven工程，配置对应的pom文件来管理项目依赖

* 常用命令：

  | 序号 |    命令     |    功能    |
  | :--: | :---------: | :--------: |
  |  1   |    clean    |    清除    |
  |  2   |    test     |  运行测试  |
  |  3   |   package   |    打包    |
  |  4   |   install   | 在本地安装 |
  |  5   | source：jar |  源码打包  |
